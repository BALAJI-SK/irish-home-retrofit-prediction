Formula (Gaussian MLE log-likelihood):
  log L = -n/2·ln(2π) - n/2·ln(RSS/n) - n/2
  AIC   = -2·log L + 2k
  BIC   = -2·log L + k·ln(n)

k definitions:
  LightGBM      : actual leaf nodes summed across all boosting trees
  Random Forest : terminal leaf nodes summed across all decision trees
  Ridge         : fitted coefficients + 1 (intercept)

Lower AIC / BIC = better fit-complexity trade-off
